# My Portfolio

My Portfolio is a modern and dynamic website built with React and Three.js, designed to showcase my education, skills, projects, and coding profiles in a visually appealing 3D format.



**1. Built with React and Three.js:** Enjoy a fast, responsive, and immersive user experience with modern 3D visuals.

**2. About Me:** Learn about my background, interests, and professional journey.

**3. Education:** Detailed information about my academic qualifications and institutions attended.

**4. Coding Profiles:** Links to my coding profiles on GeeksforGeeks, LeetCode, Codeforces, and CodeChef.

**5. Projects:** Showcase of my major projects, including descriptions, technologies used, and live demos or repositories.

**6. Skills:** Overview of my technical skills and proficiencies.

**7. Contact Me:** Easily get in touch with me through the contact form.

**Live Demo:** Check out my portfolio in action [here](https://portfolio-14yrtq7uq-mayank1803s-projects.vercel.app).
